# E-Commerce Sales and Customer Behavior Analysis

Overview

This project analyzes e-commerce sales data to uncover meaningful patterns, trends, and insights using Python and Power BI. It involves data cleaning, exploratory data analysis (EDA), visualization, and dashboard creation.

Objectives
- Identify patterns and trends in sales data
- Understand customer behavior
- Analyze product and regional performance
- Provide data-driven business insights

Dataset
* Source: Kaggle
* Records: 34,000+ transactions

Features Include:
- Product Category
- Price, Quantity, Total Amount
- Discount, Shipping Cost
- Customer Age, Gender
- Payment Method
- Region
- Order Date

Tools & Technologies
1. Python – Data Analysis
2. Pandas & NumPy – Data Manipulation
3. Matplotlib & Seaborn – Visualization
4. Power BI – Dashboard & Reporting

Analysis & Features
* Data Cleaning & Preprocessing
* Handling Missing Values
* Outlier Detection & Treatment (Clipping)
* Feature Engineering (Month & Year extraction)

EDA Techniques Used:
1. Univariate Analysis → Histogram, Countplot, Pie Chart
2. Bivariate Analysis → Scatter Plot, Line Chart, Bar Chart, Box Plot
3. Multivariate Analysis → Heatmap, Pairplot

Key Insights
* Electronics is the top-performing category
* Sales show consistent trends across months
* Higher product prices lead to higher transaction values
* Credit cards are the most preferred payment method
* Regional performance is balanced across all regions

Project Outcome

This project demonstrates how raw e-commerce data can be transformed into actionable insights using data analysis and visualization techniques. It showcases the integration of Python-based EDA with Power BI dashboards for effective business storytelling.
