# E-Commerce Sales and Customer Behavior Analysis

Overview

This project analyzes e-commerce sales data to uncover key patterns, trends, and insights. It includes data cleaning, exploratory data analysis (EDA), and visualization using Python, along with interactive dashboard creation in Power BI.

The goal is to understand:

Sales performance
Customer behavior
Product trends
Regional insights

Dataset
Source: Kaggle
Records: 34,000+ transactions
Features include:
Product Category
Price, Quantity, Total Amount
Discount, Shipping Cost
Customer Gender, Age
Payment Method
Region
Order Date

| Tool       | Purpose                |
| ---------- | ---------------------- |
| Python     | Data Analysis          |
| Pandas     | Data Manipulation      |
| NumPy      | Numerical Operations   |
| Matplotlib | Visualization          |
| Seaborn    | Advanced Visualization |
| Power BI   | Dashboard & Reporting  |

Analysis & Features

✔ Data Cleaning & Preprocessing
✔ Handling Missing Values
✔ Outlier Detection & Treatment (Clipping)
✔ Feature Engineering (Month, Year extraction)
✔ Univariate Analysis
✔ Bivariate Analysis
✔ Multivariate Analysis
✔ Data Visualization using multiple chart types
✔ Interactive Power BI Dashboards with KPI Cards & Slicers

Key Insights
📊 Electronics is the top-performing category
📈 Sales remain consistent across months
💰 Higher prices → higher transaction values
💳 Credit cards are the most preferred payment method
🌍 Regional performance is balanced across locations

Project Outcome

This project transforms raw e-commerce data into meaningful insights using data analysis and visualization techniques. It demonstrates the ability to combine Python-based EDA with Power BI dashboards for effective business storytelling and decision-making.
